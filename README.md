# Pong Game
![screenshot](/pong.PNG "inhabitent homepage screenshot")
A basic pong game using SVGs.

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* w: up
* s: down

**Player 2:**
* ▲ : up
* ▼: down
